Layout & Flexbox

This lesson covers how to arrange elements on the page — side by side, centered, spaced out, and more.

The Display Property

Every HTML element has a default display value. The two most common ones are:

  • block — takes up the full width, stacks vertically (like <div>, <h1>, <p>)
  • inline — only as wide as its content, sits side by side (like <span>, <a>, <strong>)
Block element — takes full width
Another block — goes below
Inline elements sit side by side 
div  { display: block; }   /* default for div */
span { display: inline; }  /* default for span */

What is Flexbox?

Flexbox is a layout system that makes it easy to arrange items in a row or column.

You turn it on by adding display: flex; to a parent container. The children inside automatically line up.

Item B
Item C
.container {
  display: flex;
}

That's it — one line turns stacked blocks into a row.

flex-direction (Row vs Column)

By default, flex items go in a row (left to right). You can change them to a column (top to bottom).

Centering (The Holy Grail)

Centering something both horizontally and vertically used to be hard. Flexbox makes it easy:

Perfectly centered
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}

This 3-line combo is one of the most useful patterns in CSS.

Quick Reference

Here are the CSS properties you learned today:

  • display: flex; — turns on Flexbox for a container
  • justify-content — horizontal spacing (center, space-between, etc.)
  • align-items — vertical alignment (center, flex-start, flex-end)
  • flex-direction — row (default) or column
  • flex-wrap — wrap items to the next line
  • gap — even spacing between flex items

And these patterns:

  • Center anything: display: flex; justify-content: center; align-items: center;
  • Navbar layout: display: flex; justify-content: space-between; align-items: center;
  • Wrapping grid: display: flex; flex-wrap: wrap; gap: 10px;
