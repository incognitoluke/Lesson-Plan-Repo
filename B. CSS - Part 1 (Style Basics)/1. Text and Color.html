CSS Basics — Text & Color

This lesson covers how to add CSS to your HTML and change colors, fonts, and text styling.

Where Does CSS Go?

CSS lives inside a <style> tag in the <head> section of your HTML file.

Think of it this way: HTML is the skeleton, and CSS is the outfit.

<head>
  <style>
    h1 {
      color: blue;
    }
  </style>
</head>

Anatomy of a CSS Rule

Every CSS rule has three parts:

  1. Selector — which element to style (like h1 or p)
  2. Property — what to change (like color or font-size)
  3. Value — what to change it to (like red or 24px)
selector {
  property: value;
}

Text Color

This text uses rgb(46, 204, 113).

  color: tomato;

Background Color

The background-color property fills the background of an element.

This div has a blue background.

This div has a yellow background. 
div {
  background-color: #3498db;
  color: white;
}

Font Size

Use font-size to make text bigger or smaller.

This is 12px — small text.

This is 16px — the default browser size.

This is 24px — bigger text.

This is 36px — heading size.

p {
  font-size: 24px;
}

Font Family

Use font-family to change the typeface.

This is Arial (sans-serif).

This is Georgia (serif).

This is Courier New (monospace).

body {
  font-family: Arial, sans-serif;
}

Tip: always end with a generic family like sans-serif as a fallback.

Font Weight

Use font-weight to make text bold or light.

This is light (300).

This is normal (400).

This is bold (700).

  font-weight: bold;
}

p {
  font-weight: normal;
}

Quick Reference

Here are the CSS properties you learned today:

  • color — text color
  • background-color — background fill
  • font-size — text size (use px)
  • font-family — typeface (Arial, Georgia, etc.)
  • font-weight — bold or normal
  • text-align — left, center, right
  • text-decoration — underline, strikethrough, or none

And these concepts:

  • <style> — where CSS goes (in the <head>)
  • Selectors — pick elements by tag name (h1, p, div)
  • property: value; — how every CSS rule is written
