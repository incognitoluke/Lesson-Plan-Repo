Hover, Shadows & Extras

This lesson covers interactive effects — making elements respond to the mouse with hover states, shadows, transitions, and more.

The :hover Pseudo-Class

Add :hover after any selector to apply styles only when the mouse is over it.

button {
  background: #3498db;
  color: white;
}

button:hover {
  background: #2980b9;
}

You can change any property on hover — color, background, size, border, anything.

Box Shadow

Use box-shadow to add a drop shadow behind any element.

The format is: box-shadow: x-offset y-offset blur color;

Subtle
Medium
Strong
.subtle { box-shadow: 2px 4px 8px rgba(0,0,0,0.1); }
.medium { box-shadow: 0 6px 20px rgba(0,0,0,0.2); }
.strong { box-shadow: 0 12px 40px rgba(0,0,0,0.3); }

Tip: rgba(0,0,0,0.15) means black at 15% opacity. Lower number = lighter shadow.

Transitions (Smooth Changes)

Without transition, hover changes are instant. Adding it makes things animate smoothly.

Hover me — I'm smooth!
.box {
  background: #e74c3c;
  transition: all 0.3s;
}

.box:hover {
  background: #2ecc71;
  transform: scale(1.05);
  border-radius: 20px;
}

transition: all 0.3s; means "animate all property changes over 0.3 seconds."

Cursor

Use cursor to change what the mouse pointer looks like when hovering.

pointer (hand)
crosshair
not-allowed
grab
text
button { cursor: pointer; }
.disabled { cursor: not-allowed; }

Always use cursor: pointer; on clickable things so users know they can click.

Opacity

Use opacity to make an element see-through.

opacity: 1
opacity: 0.6
opacity: 0.3
.faded {
  opacity: 0.5;
}

/* Nice hover pattern: */
img {
  opacity: 0.7;
  transition: opacity 0.3s;
}

img:hover {
  opacity: 1;
}

Range: 0 (invisible) to 1 (fully visible).

Putting It All Together

Here's a button using everything from this lesson — hover, shadow, transition, and cursor.

.mega-btn {
  background: #6c5ce7;
  color: white;
  border: none;
  padding: 16px 40px;
  font-size: 18px;
  border-radius: 8px;
  cursor: pointer;
  box-shadow: 0 4px 14px rgba(108, 92, 231, 0.4);
  transition: all 0.3s;
}

.mega-btn:hover {
  background: #5b4cdb;
  box-shadow: 0 8px 24px rgba(108, 92, 231, 0.6);
  transform: translateY(-2px);
}

Quick Reference

Here are the CSS properties you learned today:

  • :hover — styles that apply when the mouse is over an element
  • box-shadow — adds a drop shadow (x, y, blur, color)
  • transition — smoothly animates changes (use all 0.3s)
  • cursor — changes the mouse pointer (pointer, not-allowed, grab)
  • opacity — make elements see-through (0 to 1)
  • transform: scale() — makes elements bigger/smaller
  • transform: translateY() — moves elements up/down
