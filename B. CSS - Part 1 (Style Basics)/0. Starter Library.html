Where CSS Goes <style>

Setup
Preview

This text is red and bold because CSS told it to be.

Without CSS, every page looks the same — plain black text on white.

Code
/* CSS goes inside a <style> tag in <head> */ <head> <style> p { color: red; font-weight: bold; } </style> </head>
CSS = "Cascading Style Sheets." It controls how HTML looks — colors, sizes, spacing, layout.

Selectors h1, p, div

Setup
Preview

This h2 is purple

This paragraph is blue

So is this one — same rule hits all <p> tags

Code
/* Target elements by their tag name */ h2 { color: purple; } p { color: blue; }
A selector picks which elements get styled. Tag selectors style ALL elements of that type.

Class & ID .class #id

Setup
Preview

This has class="highlight"

This paragraph has no class — unaffected.

This has id="page-title"

Code
/* Class — reusable, starts with a dot */ .highlight { color: green; font-weight: bold; } /* ID — unique, starts with a hash */ #page-title { color: red; }
TypeHTMLCSS
classclass="highlight".highlight { }
idid="page-title"#page-title { }
Classes can be reused on many elements. IDs should be unique — only one per page.

Line Height line-height

Text
Preview

Line height 1.0 — the lines are cramped together and hard to read when there is a lot of text.

Line height 1.8 — much more breathing room between lines, which makes paragraphs easier to read.

Code
.tight { line-height: 1.0; } .relaxed { line-height: 1.8; }
A number with no unit (like 1.6) is a multiplier of the font size. 1.5–1.8 is good for body text.

Padding padding

Box Model
Preview
padding: 5px (small)
padding: 20px (medium)
padding: 40px (large)
Code
/* All four sides */ .box { padding: 20px; } /* Top/Bottom Left/Right */ .box { padding: 10px 20px; } /* Individual sides */ .box { padding-top: 10px; padding-right: 20px; padding-bottom: 10px; padding-left: 20px; }
Padding = space INSIDE the box, between the content and the border.

Margin margin

Box Model
Preview
margin: 0px
margin: 20px
Code
/* All four sides */ .box { margin: 20px; } /* Top/Bottom Left/Right */ .box { margin: 10px 20px; } /* Center a block element */ .box { margin: 0 auto; }
Margin = space OUTSIDE the box, pushing other elements away. auto centers it.

Concept

CSS Units px, %, em, rem

Reference
Preview

50% (of parent)

Code

UnitMeaning
pxFixed pixels — always the same size
%Percentage of the parent element
emRelative to the element's font size
remRelative to the root (<html>) font size

Start with px and %. Learn rem when you're comfortable.
