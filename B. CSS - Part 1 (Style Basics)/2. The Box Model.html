The Box Model

This lesson covers how every HTML element is a box — and how to control its size, spacing, and borders.

The Big Idea

Every single element in HTML is an invisible rectangle. CSS lets you control four layers of that rectangle:

  1. Content — the text or image inside
  2. Padding — space between the content and the border
  3. Border — the visible edge of the box
  4. Margin — space outside the border, pushing other elements away

Content

Width & Height

Use width and height to set the size of the content area.

200px wide, 80px tall

100% width (fills the parent) 
.box {
  width: 200px;
  height: 80px;
}

.full {
  width: 100%;
}

Padding

Padding adds space inside the box — between the content and the border.

padding: 5px (small)
padding: 20px (medium)
padding: 40px (lots of breathing room) 
/* All four sides */
.box { padding: 20px; }

/* Top/Bottom and Left/Right */
.box { padding: 10px 20px; }

/* Each side individually */
.box {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 10px;
  padding-left: 20px;
}

Margin

Margin adds space outside the box — pushing other elements away.

margin: 0 (no outside space)
margin: 30px (pushed away from everything) 
/* All four sides */
.box { margin: 20px; }

/* Top/Bottom and Left/Right */
.box { margin: 10px 20px; }

/* Center a block element horizontally */
.box {
  width: 300px;
  margin: 0 auto;
}

Tip: margin: 0 auto; is the classic trick to center a block element.

Border

The border property draws a visible edge around the box. It has three parts: width, style, and color.

2px solid black
3px dashed red
4px dotted blue
border-bottom only 
/* Shorthand: width style color */
.box { border: 2px solid black; }

/* One side only */
.box { border-bottom: 3px solid green; }

Border Radius

Use border-radius to round the corners of any box.

0px
12px
50%
.sharp  { border-radius: 0; }
.round  { border-radius: 12px; }
.circle { border-radius: 50%; }

Tip: Use 50% on a square element to make a perfect circle.

Quick Reference

Here are the CSS properties you learned today:

  • width / height — size of the content area
  • padding — space inside the box
  • margin — space outside the box
  • border — visible edge (width + style + color)
  • border-radius — rounds the corners

And these shortcuts:

  • padding: 10px 20px; — top/bottom, left/right
  • margin: 0 auto; — center a block element
  • border-radius: 50%; — makes a circle
  • border-bottom: — style just one side
