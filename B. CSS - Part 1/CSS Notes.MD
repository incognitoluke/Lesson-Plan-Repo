### CSS Overview:

CSS is "Cascading Style Sheets"

If HTML is the skeleton and bones, CSS is the flesh pores and skin tone giving the asthetics

Basis of all CSS:
- Three different types of CSS methods: inline styles, global styles and css files
- CSS like HTML is also static, does not change based on user input alone (except some basic effects)
- CSS uses classes and IDs to distinguish different types of elements and apply the styling accordingly
