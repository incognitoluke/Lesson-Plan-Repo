# 🎨 CSS Project: Style Your Restaurant Menu Page

## Overview

**Do it yourself:** Take the restaurant menu page you built in the first HTML project and turn it into a polished, styled website using CSS.

In the first project, the goal was to build the page structure with HTML: headings, paragraphs, image, menu table, hours list, order form, Google Maps iframe, and footer. In this project, you will keep that same restaurant website, but now you will make it look professional with colors, fonts, spacing, layout, cards, hover effects, and shadows.

---

## What You're Building

Start with your completed `index.html` restaurant project. Add CSS inside the `<style>` tag in the `<head>` of the same file.

Your final page should look like a real restaurant website with:

1. **A styled hero/header section** for the restaurant name and tagline
2. **Consistent colors and fonts** across the page
3. **Clean spacing** using margin and padding
4. **Rounded cards or sections** for content areas
5. **A styled menu table** that is easy to read
6. **A styled order form** with readable inputs and labels
7. **A layout using Flexbox** somewhere on the page
8. **Hover effects** on links, buttons, or cards
9. **A footer** that matches the rest of the design

---

## Starter Rule

Do **not** delete the HTML requirements from the first restaurant project. This project builds on top of them.

You should still have:

- Restaurant name and tagline
- About section
- Image
- Menu table
- Hours list
- Order form
- Google Maps iframe

## CSS Requirements Checklist

Use this list to make sure your page includes the CSS skills from the module.

### CSS Setup
- [ ] Add a `<style>` tag inside the `<head>`
- [ ] Write CSS rules using selectors like `body`, `h1`, `p`, `table`, `button`, and classes
- [ ] Use at least **3 class selectors** such as `.hero`, `.menu-section`, `.order-form`
- [ ] Use comments in your CSS to label sections

### Text & Color
- [ ] Change the page `background-color`
- [ ] Change the main text `color`
- [ ] Style the `h1` with a different color and larger `font-size`
- [ ] Use `font-family` on the `body`
- [ ] Use `font-weight` on at least one heading or special menu item
- [ ] Use `text-align` somewhere, such as centering the header
- [ ] Remove the underline from at least one link using `text-decoration: none;`

### Box Model
- [ ] Use `padding` on major sections
- [ ] Use `margin` to create space between sections
- [ ] Use `border` on at least one section, card, image, table, or form
- [ ] Use `border-radius` to round corners
- [ ] Set a `width` or `max-width` for the main page content
- [ ] Center the main content using `margin: 0 auto;`

### Layout & Flexbox
- [ ] Use `display: flex;` at least once
- [ ] Use `justify-content` at least once
- [ ] Use `align-items` at least once
- [ ] Use `gap` to create space between flex items
- [ ] Use Flexbox for one of these:
  - Navigation links
  - Menu category cards
  - Hours and location side-by-side
  - Form inputs side-by-side

### Tables, Forms & Images
- [ ] Style the menu `table` with width, padding, and borders
- [ ] Style `th` differently from `td`
- [ ] Style form `input`, `select`, and `textarea` fields
- [ ] Make the submit `button` look clickable
- [ ] Style the image with width, border-radius, or border
- [ ] Style the Google Maps `iframe` with width and border-radius

### Hover, Shadows & Extras
- [ ] Add `:hover` to at least one link or button
- [ ] Use `cursor: pointer;` on the submit button
- [ ] Add `box-shadow` to at least one section, card, image, or form
- [ ] Use `transition` so a hover effect changes smoothly
- [ ] Optional: use `opacity` or `transform` on hover
