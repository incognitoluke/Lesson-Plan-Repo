# 🍕 Project: Build a Restaurant Menu Page

## Overview

**Do it yourself:** Build a one-page website for a made-up restaurant. You pick the name, the food, the vibe — everything. The only rule is you have to use the HTML elements listed below.

---

## What You're Building

A single `index.html` file that looks like a real restaurant's website. It should have:

1. **A restaurant name and tagline** at the top
2. **A short "About Us" paragraph** describing the restaurant
3. **An image** (a photo of food, the restaurant, anything that fits)
4. **A menu** displayed in a table with item names, descriptions, and prices
5. **A list of hours** the restaurant is open
6. **An order form** where a customer can place a takeout order
7. **Google Maps** add a location iframe with the actual location
8. **A footer** with a link to the restaurant's (fake) social media

---

## Requirements Checklist

Use this list to make sure you've included everything. Check them off as you go.

### Page Structure
- [ ] Your file starts with `<!DOCTYPE html>`
- [ ] You have `<html>`, `<head>`, `<body>` tags
- [ ] You have a `<title>` in the `<head>`

### Text & Content
- [ ] At least one `<h1>` for the restaurant name
- [ ] At least two `<h2>` tags for section titles (like "Our Menu", "Hours", "Order Online")
- [ ] At least two `<p>` tags
- [ ] Use `<strong>` or `<em>` somewhere (bold a special dish, italicize a tagline, etc.)
- [ ] Use `<hr>` to separate sections

### Links & Images
- [ ] At least one `<img>` with a `src` and `alt` attribute
- [ ] At least one `<a>` link (can go to a real site or just `#`)

### Lists
- [ ] One `<ul>` (unordered list) — maybe for hours, toppings, or fun facts
- [ ] One `<ol>` (ordered list) — maybe for "Top 3 Sellers" or steps to order

### Table
- [ ] A `<table>` with `<thead>` and `<tbody>`
- [ ] At least 3 columns (Item, Description, Price)
- [ ] At least 4 rows of menu items

### Form & Inputs
- [ ] A `<form>` tag wrapping the order section
- [ ] An `<input type="text">` for the customer's name
- [ ] An `<input type="email">` for their email
- [ ] At least one set of `<input type="radio">` buttons (pick a meal size, crust type, etc.)
- [ ] At least one `<input type="checkbox">` (add toppings, sides, etc.)
- [ ] A `<select>` dropdown (pick a drink, delivery time, etc.)
- [ ] A `<textarea>` for special instructions
- [ ] A `<button type="submit">` to place the order
- [ ] Use `placeholder` on at least two inputs
- [ ] Use `required` on at least one input

### IFrame
- [ ] An `<iframe>` element with Google Maps
