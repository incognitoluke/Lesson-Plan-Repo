Paragraph <p>

Text
Preview

This is a paragraph. It wraps text into a block with spacing above and below.

This is a second paragraph. Browsers add a gap between them automatically.

Code
<p>This is a paragraph.</p> <p>This is a second paragraph.</p>
No special attributes needed.

Text Formatting <strong> <em> <mark>

Inline
Preview

This is bold text using strong.

This is highlighted text using mark.

This is small text using small.

Blockquote <blockquote>

Text
Preview
The only way to do great work is to love what you do.
Code
<blockquote cite="https://example.com"> The only way to do great work is to love what you do. </blockquote>
AttributeDescription
citeURL of the quote source (optional)

Code <code> <pre>

Inline / Block
Preview

Use console.log() to debug.

function greet(name) {
  return "Hello, " + name;
}
Code
<!-- Inline code --> <p>Use <code>console.log()</code> to debug.</p> <!-- Code block --> <pre> function greet(name) { return "Hello, " + name; } </pre>
<code> = inline snippet. <pre> = preserves whitespace for blocks.

Ordered List <ol>

List
Preview

  2. Mix the ingredients
  3. Bake for 25 minutes
Code

Data
Preview
NameAgeCity
Alice28NYC

Code

<tr> = row, <th> = header cell, <td> = data cell

Div & Span <div> <span>

Layout
Preview
This is inside a div (block-level box).

This sentence has a red span inside it (inline).

Code
<!-- Block-level container --> <div style="background: #e8f4fd;"> Content inside a div. </div> <!-- Inline container --> <p>A <span style="color: red;">red span</span> inside.</p>
<div> = block container (full width). <span> = inline container (wraps text).

Semantic Layout Tags

Structure
Preview
<header> — Site title, logo, nav
<nav> — Navigation links
<main> — Primary page content
<section> — Grouped content
<article> — Self-contained content
<footer> — Copyright, links
Code
<header> Site title & nav </header> <nav> Navigation links </nav> <main> <section> Grouped content </section> <article> Self-contained </article> <aside> Sidebar content </aside> </main> <footer> Copyright & links </footer>
Same as <div> but with meaning — helps search engines and screen readers understand your page structure.
