Inputs & Forms

This lesson covers how to collect information from users using HTML forms.

Text Input

The most basic input. Users type text into a box.


Email Input

Just like text, but the browser checks that it looks like an email.


Password Input

The text is hidden with dots so nobody can read it.


Number Input

Only allows numbers. Has little up/down arrows.


Date Input

Opens a calendar picker in most browsers.


Color Input

Opens a color picker. Try clicking it!


Textarea

For when you need more than one line of text.


Checkboxes

Let users pick one or more options.

What toppings do you want?



Radio Buttons

Let users pick only one option from a group.

What size drink?



Dropdown Menu

A compact way to pick one option from a list.


Buttons

Buttons do things when you click them.

Putting It All Together

The <form> tag wraps everything into one group that can be submitted.

Quick Reference

Here are the input types you learned today:

  • text — basic text box
  • email — text box that checks for @
  • password — hides what you type
  • number — only numbers allowed
  • date — calendar picker
  • color — color picker
  • checkbox — pick many options
  • radio — pick one option

And these other form elements:

  • <textarea> — multi-line text box
  • <select> — dropdown menu
  • <button> — clickable button
  • <form> — wraps everything together
  • <label> — describes an input
