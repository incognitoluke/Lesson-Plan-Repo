Tables

This lesson covers how to display data in rows and columns using HTML tables.

A Basic Table

Every table starts with <table> and is built from rows and cells.

Row 1, Cell 1 Row 1, Cell 2
Row 2, Cell 1 Row 2, Cell 2

Table Rows

Each <tr> creates a new row going across.

This is row 1
This is row 2
This is row 3

Table Data Cells

Each <td> creates a cell inside a row. More cells = more columns.

Column 1 Column 2 Column 3
Apple Red $1.00
Banana Yellow $0.50

Table Headers

Use <th> instead of <td> for header cells. They show up bold and centered.

Fruit Color Price
Apple Red $1.00
Banana Yellow $0.50

Thead & Tbody

Wrap your header row in <thead> and your data rows in <tbody>. This keeps things organized.

Name Age City
Alice 28 New York
Bob 34 Los Angeles
Carol 22 Chicago

Table Footer

Use <tfoot> for a summary row at the bottom, like a total.

Item Price
Burger $8.00
Fries $3.00
Drink $2.00
Total $13.00

Spanning Columns

Use colspan to make a cell stretch across multiple columns.

Weekend Schedule
Time Saturday Sunday
Morning Soccer Sleep in
Afternoon Homework Movies

Spanning Rows

Use rowspan to make a cell stretch down multiple rows.

Quick Reference

Here are the table tags you learned today:

  • <table> — creates the table
  • <tr> — creates a row
  • <td> — creates a data cell
  • <th> — creates a header cell (bold & centered)
  • <thead> — groups the header rows
  • <tbody> — groups the data rows
  • <tfoot> — groups the footer/summary row

And these attributes:

  • border — adds visible borders (on <table>)
  • colspan — cell stretches across columns (on <td> or <th>)
  • rowspan — cell stretches down rows (on <td> or <th>)
