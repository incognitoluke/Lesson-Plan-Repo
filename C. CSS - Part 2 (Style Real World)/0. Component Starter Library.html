/* Cards wrap automatically — no media queries needed! */ .grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(260px, 1fr)); gap: 20px; }
KeywordBehavior
auto-fitFills available space, stretches items
auto-fillFills available space, keeps empty tracks

This one line replaces media queries for most card grids. It's the most useful CSS Grid trick.

Preview
What is CSS Grid?
CSS Grid is a two-dimensional layout system that lets you create rows and columns simultaneously.
When should I use Flexbox vs Grid?
Flexbox is best for one-dimensional layouts (row or column). Grid is best for two-dimensional layouts.
Can I nest grids?
Yes — a grid item can also be a grid container. This is called subgrid in newer CSS.
HTML + CSS
<details> <summary>Question?</summary> <p>Answer here.</p> </details> /* No JavaScript needed! */ details { border: 1px solid #e5e5e5; border-radius: 8px; } summary { padding: 12px 16px; cursor: pointer; font-weight: 600; }
The <details> element is built-in HTML — open/close works with zero JavaScript.
